AMD setup with Zepto

This page uses the AMD build in the dist directory, not the files in src. In other words, run the grunt command first to build a current version.

On mobile, swipe down to enter minimal UI

The page demonstrates the use of Zepto, instead of jQuery, with the AMD build of jQuery.documentSize. See require-config.js for the setup. This page won't work in IE8 and other legacy browsers.
